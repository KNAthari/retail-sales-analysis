# Retail Sales SQL Analysis

## Project Overview
This project aims to analyze retail sales data and identify trends in revenue, profit, and regional performace using Postgres SQL

## Tools Used
- Postgres SQL
- VSCode
- Git/Github

## Business Questions
This analysis explores retail performance across regions, time periods and product categories to understand revenue drivers, profitability efficiency and data quality
- Which regions contribute the most to total revenue and total profit?
- Which regions generate high revenue but relatively low profit, indicating lower profitability efficiency?
- How does transaction volume vary across regions and product categories?
- Do higher transaction volumes correlate with higher revenue or profit across regions and categories?
- Are there seasonal trends in revenue and profit across product categories such as Books and Electronics?
- How many transactions contain inconsistent or potentially invalid values (e.g., zero revenue but positive profit)?
